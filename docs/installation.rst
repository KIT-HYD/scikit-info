==================
Installation Guide
==================

.. toctree::
    :maxdepth: 2



You can install scikit-info either using the Python Package Index or
directly from GitHub.

PyPI
====

Install scikit-info using pip:

.. code-block:: bash

    pip install scikit-info


GitHub
======

There might be a more recent version on GitHub. You can download and install
the package like:

.. code-block:: bash

    git clone https://github.com/kit-hyd/scikit-info.git
    cd scikit-info
    pip install -r requirements.txt
    pip install -e .
